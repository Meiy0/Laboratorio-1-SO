Especificación:
- El MAKE tiene como nombre "makefile".
- El archivo c++ tiene como nombre "Lab1.cpp"
- Viene incluida la carpeta "archivos_deportes", el cual contiene los archivos de prueba. En caso de tener otros archivos de prueba usar el mismo nombre de la carpeta.
- El output se generará en una carpeta llamada "Carpeta actual".
- Se usó Ubuntu para pruebas.

Instrucciones: 
- Se debe tener instalado make y el compilador g++.
- Desde la terminal usar el comando "make run", este debe ejecutar el makefile.
